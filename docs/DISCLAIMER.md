## Research Prototype Only

CEH-Orbit Blockchain Fire Seed is a research prototype.

It is provided to explore how the original CEH-Orbit design can be extended into a minimal blockchain application path. It must not be interpreted as a finished cryptographic product, an audited blockchain platform, or a production security system.

## No Formal Security Guarantee

This project currently does not provide:

- formal reduction proofs
- certified security claims
- production-grade implementation guarantees
- side-channel resistance guarantees
- standard-compliance guarantees

Any statement about observed forgery resistance, low collision rate, or strict acceptance behavior should be interpreted as empirical prototype behavior only.

## No Production Use Recommendation

This repository is not recommended for:

- storing real money
- custody of high-value assets
- regulated financial environments
- safety-critical systems
- government or military production deployment

## Experimental Blockchain Layer

The blockchain layer in this repository is intentionally minimal. It includes a local demonstration of transactions, mempool behavior, Merkle construction, block assembly, simplified mining, and state updates, but it does not constitute a complete or production-grade blockchain network.
